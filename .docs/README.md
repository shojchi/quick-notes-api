## üéØ Project Overview

**Quick Notes API** is a lightweight REST API built with Nest.js for managing quick developer notes.

### Purpose

- Learn Nest.js core concepts (modules, controllers, services, DTOs)
- Practice backend API development
- Build something useful for daily development work

### MVP Scope

- Create, read, update, delete notes
- Simple search/filter functionality
- File-based storage (JSON)
- No authentication (single user)
- RESTful API design

### Tech Stack

- **Runtime:** Node.js v18+
- **Framework:** Nest.js 10+
- **Language:** TypeScript 5+
- **Storage:** JSON file (for MVP, database later)
- **Validation:** class-validator
- **Documentation:** Swagger/OpenAPI
